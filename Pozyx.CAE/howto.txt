-------------------
How to debug & run:
-------------------

All 64 bit.

non-AMP:

	start project: Pozyx.CAE.TestApp

	Debug => Debug F5

	Run => Release CTRL+F5

AMP:

	start project: Pozyx.CAE.Lib.AMP
	rebuild when .NET changes!

	Debug => Debug F5
		
		Auto+Warp: C++ CPU and GPU
		Mixed+Warp: .NET CPU and C++ GPU

	Run => Release CTRL+F5
